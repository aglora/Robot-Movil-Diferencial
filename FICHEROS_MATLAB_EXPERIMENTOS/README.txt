En esta carpeta se encuentran los ficheros .txt de datos experimentales y los scripts funciones de Matlab para la representación de las gráficas a partir de ellos. 

Es necesario trabajar en el mismo directorio con todos estos archivos.

Para representar las gráficas de los modos 1 al 6 se usa telemetriaM1_M6.m

Para representar las gráficas del modo 7 se usa telemetriaM7.m

Para representar las gráficas del proyecto (modo 8) se usa telemetriaM8_carril.m, telemetriaM8_obstaculos.m y telemetriaM8_puntos.m en función del experimento con el nombre asociado a los mismos.
